                       GETTING STARTED GUIDE
================================================================================

1. LOCAL DEVELOPMENT:
   $ npm install
   $ npx prisma generate
   $ npm run dev
   Access: http://localhost:3000

2. PRODUCTION SETUP:
   - Read PRODUCTION.md for comprehensive guide
   - Set environment variables (AUTH_SECRET, DATABASE_URL, etc.)
   - Configure AI providers in admin dashboard
   - Run database migrations
   - Deploy to Vercel or Docker

3. FIRST STEPS:
   - Go to http://localhost:3000
   - Register new user account
   - Create a chat
   - Send a message to AI
   - Access admin at /admin

   - Set DATABASE_URL to your database

   - Run: npx prisma db push
   - Run: npx prisma generate

   - Run: npm run dev
   - Visit: http://localhost:3000
   - Test registration and chat

4. ADD AI PROVIDERS:
   - Register user (become admin if ADMIN_TOTP_SECRET set)

   - Test health endpoint
